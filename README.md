## 介绍
这个示例项目是为了帮助使用AVOSCloud的开发者快速实现具有实时通讯功能的应用.

## [Leanchat 项目构成](https://github.com/leancloud/leanchat-android/wiki)

## 运行

1.  `pod install`       
2.  用XCode打开`AVOSChatDemo.xcodeproj`
